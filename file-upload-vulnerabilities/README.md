# file-upload-vulnerabilities

## 📌 مقدمة
هذا المستودع يشرح ثغرات رفع الملفات (File Upload Vulnerabilities) في تطبيقات الويب، مع تحليل لطرق الاستغلال الشائعة، وأساليب التجاوز (Bypass Techniques)، وتأثير كل نوع من الثغرات، بالإضافة إلى أفضل الممارسات الأمنية للوقاية منها.

---

## 🎯 الهدف من المشروع
- فهم كيف تعمل آلية رفع الملفات في تطبيقات الويب
- التعرف على أخطاء التصميم الأمني في هذه الخاصية
- دراسة طرق الاستغلال عملياً
- تعلم أساليب الحماية الصحيحة

---

## ⚠️ خطورة ثغرات رفع الملفات
ثغرات رفع الملفات قد تؤدي إلى:
- تنفيذ أوامر عن بعد (Remote Code Execution)
- رفع Web Shell والسيطرة على السيرفر
- تخزين ملفات خبيثة داخل النظام
- تجاوز صلاحيات التطبيق
- استغلال الثغرة كجزء من سلسلة هجمات

---

# 🧩 أنواع ثغرات رفع الملفات

## 1. ❌ رفع بدون تحقق (No Validation)
### الوصف
التطبيق يسمح برفع أي ملف بدون أي فحص.

### التأثير
- رفع ملفات تنفيذية مباشرة مثل:
  - .php
  - .jsp
  - .asp

### النتيجة
تنفيذ أوامر على السيرفر مباشرة (RCE).

---

## 2. ⚠️ تحقق ضعيف (Client-Side Validation)
### الوصف
التحقق يتم في واجهة المستخدم فقط أو بطريقة غير قوية مثل:
- فحص الامتداد فقط
- الاعتماد على Content-Type

### طرق التجاوز
- تعديل الطلب عبر Burp Suite:
  - تغيير Content-Type
- تغيير الامتداد:
  - shell.php.png
  - shell.jpg.php

### المشكلة
التحقق في المتصفح فقط وليس في السيرفر.

---

## 3. 📂 رفع إلى مجلد غير قابل للتنفيذ
### الوصف
الملف يتم رفعه لكن داخل مجلد لا يتم تنفيذ الكود فيه.

مثال:
/uploads/

### الاستغلال
يحتاج إلى:
- ثغرة LFI
- أو سوء إعدادات السيرفر

---

## 4. 🚫 التصفية السوداء (Blacklist Filtering)
### الوصف
منع امتدادات معينة مثل:
.php, .jsp, .asp

### طرق التجاوز
- امتدادات بديلة:
  - .php5
  - .phtml
  - .phar
- تغيير حالة الأحرف:
  - .PhP
- امتدادات مزدوجة:
  - shell.php.jpg

### المشكلة
الـ blacklist دائمًا غير مكتمل.

---

## 5. ⚙️ استغلال إعدادات السيرفر
### الوصف
رفع ملفات تؤثر على إعدادات السيرفر نفسه.

### أمثلة

Apache (.htaccess):
AddType application/x-httpd-php .jpg

IIS (web.config):
تعديل سلوك تنفيذ الملفات داخل السيرفر.

### التأثير
تحويل ملفات غير تنفيذية إلى ملفات قابلة للتنفيذ.

---

## 6. 🧬 إخفاء الامتداد (Obfuscation)
### الطرق
- ترميز الأحرف
- امتدادات مزدوجة
- Null byte (في أنظمة قديمة)
- امتدادات بديلة:
  - .php5
  - .phtml

### الهدف
تجاوز فلاتر تعتمد على تحليل بسيط لاسم الملف.

---

## 7. ⏱️ Race Condition
### الوصف
وجود فرق زمني بين:
1. رفع الملف
2. فحصه
3. نقله أو تنفيذه

### الاستغلال
- رفع ملف خبيث
- الوصول إليه قبل الحذف أو المعالجة

---

## 🛡️ طرق الحماية (Mitigation)

- التحقق من السيرفر فقط
- استخدام Allowlist
- إعادة تسمية الملفات
- تخزين الملفات خارج Web Root
- تعطيل تنفيذ الملفات في upload directory
- فحص محتوى الملف وليس الاسم فقط
- استخدام WAF

---

## 📌 الخلاصة
ثغرات رفع الملفات تعتمد على فشل في التحقق والتخزين والتنفيذ داخل السيرفر.
